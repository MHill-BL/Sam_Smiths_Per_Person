# Sam\_Smiths\_Per\_Person



\# Sam Smith's Pubs Per Capita Analysis



Analysing the density of Samuel Smith's pubs across Yorkshire towns and cities.



\## About



Have you ever wondered which place has the most Sam Smiths per person in Yorkshire? Well now you can answer that question. As you all know Sam Smiths open and close all the time so this information may not be up to date. I did my best to find if each pub is open or not.





\## Data Sources



\- \*\*Pubs data\*\*: \[Brewery History Society Wiki](http://breweryhistory.com/wiki/)

\- \*\*Population data\*\*: \[City Population](https://www.citypopulation.de/en/uk/yorkshireandthehumber/) 



\## Key Findings



\- Follifoot has the most with 331 people per Sam Smiths pub. But the village is small with only 2 pubs.

\- Tadcaster, the home of Sam Smiths, is 2nd.

\- Large cities and towns do badly with Huddersfield, Sheffield, Harrogate and Bradford the bottom 4.

\- Leeds does better as it has a high number of Sam Smiths.
